Build a premium, visually clean “Mentorship Tracks” page for Designforge.

Brand context:
Designforge is a mentorship-led design learning ecosystem for aspirants, students, and early professionals.

Important:
This page should feel consistent with the other Designforge pages already created:
- premium
- warm
- editorial
- concise
- mixed UI patterns
- not card-heavy
- easy to scan

Page goal:
Help users understand and choose between the two main mentorship directions:
1. Admissions Mentorship
2. Job Preparation

Do not repeat too much content from the Focus Batch, About, or Apprenticeship pages.
This page should work more like a clear pathway selection page.

Tone:
- mentorship-led
- credible
- concise
- structured
- aspirant-first
- not like a generic coaching website

Use these UI patterns:
- split hero
- statement strip
- editorial 2-column section
- 2-track split layout
- tabbed or stacked admissions section
- editorial job-prep section
- 4-step process timeline
- audience band/list
- centered CTA banner

Avoid:
- too many repeated cards
- long paragraphs
- repeating broad brand story
- repeating too much exam-prep detail already covered elsewhere
